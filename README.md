# Vehicle OOP Spring Boot Project

A simple **Spring Boot** application demonstrating **Object-Oriented Programming (OOP)** principles using **abstract classes** and **inheritance** â€” integrated with **Oracle Database**, **logging**, **exception handling**, and **unit testing**.

---

## Overview

The application manages different vehicle types:
- Car 
- Bike 
- Truck 
- Electric Car 

Each vehicle inherits from an abstract `Vehicle` class and implements its own `start()` method.
